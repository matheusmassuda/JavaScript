JS Interno e Externo

É possível escrever dentro da tag script ou referenciar com src = " " a tag script e linkar com arquivo.js externo.

A tag script vem no final a tag body pois o documento HTML é lido de cima para baixo e é preferível renderizar primeiro o HTML e CSS antes de rodar os scripts (que afetarão o HTML e CSS). Além de que é mais demorado renderizar scripts, então é indicado já deixar o usua´rio com "algun conteúdo" caso haja demora para renderizar.
